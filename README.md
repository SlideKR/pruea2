# Aysén Vivo

Aplicación web de turismo sostenible para la Región de Aysén.

## Funcionalidades

- Catálogo dinámico de experiencias.
- Filtros por categoría con botones interactivos.
- Formulario de reserva validado en JavaScript.
- Descuento de cupos al confirmar una reserva.

## Cómo ejecutar

1. Abre [index.html](index.html) directamente en el navegador, o ejecuta:
   - `python3 -m http.server 3000`
2. Visita `http://localhost:3000`.
